Your First Open Source Contribution

Contributing to open source is one of the best ways to grow as a developer, build your portfolio, and connect with the global dev community.

Step 1: Find a Good First Issue

Look for issues labeled good first issue or beginner friendly on GitHub.

Step 2: Fork and Clone

git clone https://github.com/your-fork/repo.git
cd repo

Step 3: Make Your Change

Create a new branch, make your change, and commit with a clear message.

Step 4: Open a Pull Request

Push your branch and open a PR with a clear title and description explaining what you changed and why.
